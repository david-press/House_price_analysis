# USA Housing Dataset

## 📌 Project Overview
This project analyzes different factors that can affect the prices of houses in the united states.

## 🎯 Objectives
- Perform data cleaning and preprocessing
- Conduct Feature Engineering and exploratory data analysis (EDA)
- Identify patterns affecting prices using correlation Matrix
- Generate insights regarding the datasets

## 🛠 Tools Used
- Python
- Pandas (Data Analysis)
- Matplotlib (Visualization)
- Seaborn (Visulzation)
- numpy (numerical maniplation and conversion)


## 📊 Key Analysis Performed
- Feature Engineering (price_group)
- Feature correlation matrix 
- Distribution analysis (skew and outliers)
- Multi-variable grouping

## 🧠 Conclusion
The dataset shows no strong linear relationships with Price, with only one feature exhibiting moderate correlation and the rest weak. 
This suggests that house prices are not driven by a single dominant variable but rather a combination of multiple factors. 
As a result, relying solely on correlation-based feature selection would be insufficient, and more advanced methods such as feature interactions, 
transformations, or non-linear models will be necessary.
